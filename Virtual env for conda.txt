Step 1:

To create virtual environment in conda
-----------------------------------------------
$ conda create --name YT-env python=3.11
-----------------------------------------------

Step 2:

To activate virtual env
-----------------------------------------------
$ conda activate YT-env
-----------------------------------------------
OR
To Deactive virtual env
-----------------------------------------------
$ conda deactivate
-----------------------------------------------

Step 3:

Activate the Kernal you want to work on 
--------------------------------------------------------------
ipython kernel install --user --name=myenv --display-name "YT-env"
--------------------------------------------------------------


Now you can install all the packages

Eg. conda install pandas
